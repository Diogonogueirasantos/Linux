Este repositório foi criado com intuito de registrar toda minha jornada com sistemas que utilizam kernel Linux. Apesar de não ter nada aqui voltado para uma única **distro**, todos meus estudos e vivencia utilizados Linux foi e é voltado para a família **RPM**, mais especificamente a distro **Fedora**.

Este repositório foi divido em quatro sessões, cada uma voltada a uma área especifica de atuação, estas são: Fundamentos_Linux, Performance_Linux, LPIC_1, Redes_com_Linux.

Para cada uma dessas sessões temos uma página notion, e lá que destino toda a explicação por volta do conteúdo que estudei, sem contar que, lá posso deixar registros de conteúdos externos, sendo estes: vídeos, blogs e artigos.
